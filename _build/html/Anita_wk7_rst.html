Background¶

This document is part of a homework assignment. The goals of the assignment are to:

  1. Become familiar with RST (reStructured Text)

  2. Create a Sphinx Project

  3. Make an account on readthedocs.

Document Goals¶

The goals of this document are to:

  • Provide very general instructions to create a Sphinx project.

  • Provide the author practice in writing in RST.

Warning

These instructions are only for Mac OS X. The author, in fact, is running OS 10.11.6, El Capitan.

Document Structure¶

This document will give you general guidance on the following processes:

  1. Writing in RST

  2. Installing Sphinx

  3. Creating a Sphinx project
